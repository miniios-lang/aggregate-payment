# 聚合支付 H5 页面\n\n支持微信支付、支付宝、云闪付的统一支付 H5 页面。\n\n## 功能特性\n\n- 统一订单管理（自动生成订单号）\n- 统一回调处理\n- 统一退款流程（原路退回）\n- 统一对账面板（分渠道统计）\n- 15分钟订单倒计时\n- 响应式移动端适配 UI\n- 支付方式动态切换（按钮颜色跟随变化）\n\n## 使用方式\n\n直接在浏览器中打开 `index.html` 即可预览。\n\n## 技术说明\n\n本页面为前端模拟，所有支付流程均在本地模拟运行，交易记录通过控制台打印回调日志。接入真实后端时，可将 `console.log` 中的回调数据替换为实际 API 调用。\n
